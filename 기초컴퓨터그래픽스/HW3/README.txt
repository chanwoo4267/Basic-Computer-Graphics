유의사항 : 
숫자, 마우스 입력을 제외한 모든 기능은 소문자 알파벳 입력으로 실행됨.
기능 3, 4, 5는 '5' 입력을 통해 5번카메라로 보고 있을때만 기능이 작동한다.
기능 6은 '1' 입력을 통해 1번 카메라로 보고 있을때만 기능이 작동한다.
프로그램 시작시 동적 물체는 전부 움직이고, 시점은 1번카메라이다.

기능 0. 

기본적으로 가상의 3차원세계의 바닥과 좌표의 기준이 되는 세상 좌표계가 표시됨.
5개의 서로 다른 정적인 물체, 4개의 동적인 물체가 배치되어있음.

기능 1. 

'v', 'b', 'n', 'm' 각 입력으로 동적인 4가지 물체의 움직임을 멈추고 다시 움직이게 할 수 있다.

기능 2. 

'1', '2', '3', '4', '5' 각 입력으로 카메라 변경 가능. 
이때 '1' 카메라는 시선방향 이동이 구현된 카메라, '2' '3' '4'는 고정된 카메라, '5' 카메라는 동적인 카메라이다.

기능 3.

'a', 'd' 입력으로 5번 카메라를 x축 방향으로 이동(translation)한다.
'w', 's' 입력으로 5번 카메라를 y축 방향으로 이동(translation)한다.
'q', 'e' 입력으로 5번 카메라를 z축 방향으로 이동(translation)한다.

기능 4. 

'f', 'g' 입력으로 5번 카메라를 x축 기준으로 회전(rotation)한다.
'h', 'j' 입력으로 5번 카메라를 y축 기준으로 회전(rotation)한다.
'k', 'l' 입력으로 5번 카메라를 z축 기준으로 회전(rotation)한다.

기능 5.

'o', 'p' 입력으로 5번 카메라를 zoom in, zoom out한다.

기능 6.

마우스 왼쪽 클릭을 하며 마우스를 움직여 1번 카메라의 시선의 방향을 바꿀 수 있다.

기능 7.

'r', 't', 'y' 입력으로 5번 카메라를 시점(바라보는 위치)를 고정한 채 x, y ,z축 방향으로 회전한다.
본 기능은 카메라 회전 요구내용이 애매해 추가적으로 구현함.
